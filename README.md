## 💻 About DOKLIB

❓왜 DOKLIB 인가요?

    ☝ 팀명인 독수리의 "DOK", 정보들을 쌓아가는 도서관의 의미로 "LIB"을 따왔습니다
    
❓DOKLIB은 어떤 서비스인가요?

    ☝ 회사, 프로젝트, 카테고리, 릴리즈노트 등을 계층적, 체계적으로 관리
    ☝ 보기 쉬운 UI와 함께 사용자 및 개발자 모두를 고려한 화면
    ☝ 리소스 보호를 위한 탄탄한 보안 시스템을 구축
